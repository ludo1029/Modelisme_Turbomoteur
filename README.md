# Portfolio
## Projet 1 – Contrôle moteur DC

Ce projet permet de contrôler la vitesse d’un **moteur DC** avec un **potentiomètre**.

### Objectif
- Lire la valeur du potentiomètre
- Réguler la vitesse via PWM sur Arduino

### Résultats
Vitesse stable, précision ±5 %
